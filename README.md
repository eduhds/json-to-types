## Instalação

```sh
npm i json-to-types
yarn add json-to-types
pnpm add json-to-types
```

## Desenvolvimento

```sh
# Limpar outputs
pnpm run clean

# Compilar lib
pnpm run tsc

# Executar testes
pnpm run test

# Gerar documentação
pnpm run docs
```
